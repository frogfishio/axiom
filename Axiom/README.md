# Axiom v0.1 — Data-Centric Language Spec (Draft)

This archive contains a detailed specification for **Axiom**, a minimalist yet extremely powerful
data-centric language built on the semantics of **Structured Data Algebra (SDA)** plus:

- **Enrichment** via explicit, versioned **Sources** (pure given inputs)
- **PIC** binary codecs with **OCCURS** and **REDEFINES/overlay** (COBOL love, but algebraic)
- Stable error codes and conformance requirements
